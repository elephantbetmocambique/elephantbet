Como partilhar o acesso

Sem Google Sheets
1. Adicione o utilizador nesta página
2. O utilizador entra com o login e PIN definidos
3. Partilhe o ficheiro HTML por pen, Drive ou e-mail
⚠ Os dados não persistem entre sessões sem Sheets.
Com Google Sheets (recomendado)
1. Configure o Google Sheets (ver Guia)
2. Adicione o utilizador aqui — fica gravado no Sheet
3. Partilhe o URL ou ficheiro HTML com o utilizador
4. O utilizador selecciona o nome na lista de login e entra com o PIN
✓ Acesso em qualquer dispositivo, dados sempre actualizados.

Sobre o formato CSV

O ficheiro CSV pode ser aberto directamente no Microsoft Excel, Google Sheets ou LibreOffice Calc. O ficheiro inclui BOM UTF-8 para garantir compatibilidade com caracteres especiais (ã, ç, é...). Se o Google Sheets estiver ligado, pode também exportar directamente a partir do servidor usando o URL do Apps Script com ?action=exportCSV.

Guia de Integração — Google Sheets (custo zero)

Esta integração usa o Google Apps Script como backend gratuito. Os dados ficam guardados no Google Sheets e acessíveis por toda a equipa em tempo real, sem nenhum custo de servidor.
1

Criar o Google Sheet

Aceda a sheets.google.com com a conta Google da ElephantBet e crie uma nova folha de cálculo. Dê o nome "ElephantBet_Agentes". Não precisa de criar as abas manualmente — o script faz isso automaticamente no passo 4.

2

Copiar o ID da folha

No URL do Google Sheet, copie o ID que aparece entre /d/ e /edit:

https://docs.google.com/spreadsheets/d/SEU_ID_AQUI/edit
3

Abrir o Apps Script e colar o código

No Google Sheets clique em Extensões → Apps Script. Apague tudo e cole o código do ficheiro ElephantBet_AppsScript.gs (disponível para download junto com este sistema). Substitua COLE_AQUI_O_ID_DO_SEU_GOOGLE_SHEET pelo ID copiado no passo 2.

4

Executar a função de inicialização

No editor do Apps Script, no menu de funções, seleccione inicializar e clique em Executar. Isto cria automaticamente as abas "Agentes", "Supervisores" e "Log" com os cabeçalhos correctos e os supervisores iniciais.

5

Publicar como Web App

Clique em Implementar → Nova implementação. Configure assim:

Tipo: Web App Executar como: Eu (conta Google da ElephantBet) Quem tem acesso: Qualquer pessoa

Clique Implementar, autorize as permissões e copie o URL gerado (começa com https://script.google.com/macros/s/...).

6

Ligar ao sistema ElephantBet

Cole o URL do Web App na barra de sincronização no topo desta página e clique Ligar. O sistema passa a guardar todos os dados no Google Sheets automaticamente — adições, edições e consultas sincronizadas em tempo real.

7

Definir PINs reais dos supervisores

Após a inicialização, abra a aba "Supervisores" no Google Sheet e altere os PINs da coluna C para os PINs reais de cada supervisor. Cada supervisor pode também alterar o seu próprio PIN dentro do sistema usando o botão "Alterar PIN" no topo.

8

Partilhar o sistema com a equipa

Este ficheiro HTML funciona em qualquer browser sem instalação. Pode ser partilhado por:

• Google Drive — partilhe o link directo do ficheiro HTML • GitHub Pages — upload gratuito, URL fixo e permanente • Servidor local da empresa — se existir infraestrutura IT • Pen USB — funciona offline (dados em demo sem Sheets ligado)
