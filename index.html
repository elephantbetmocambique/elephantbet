Guia — Integração Google Sheets

Esta integração guarda todos os dados no Google Sheets gratuitamente. Os dados ficam acessíveis a toda a equipa em tempo real.
1

Criar o Google Sheet

Aceda a sheets.google.com e crie uma nova folha chamada "ElephantBet_Agentes".

2

Abrir o Apps Script

No Google Sheets, clique em Extensões → Apps Script. Cole o código do ficheiro ElephantBet_AppsScript.gs e substitua o SHEET_ID pelo ID da sua folha.

3

Executar a inicialização

No editor Apps Script, seleccione a função inicializar e clique Executar. Cria as abas automaticamente.

4

Publicar como Web App

Implementar → Nova implementação → Web App Executar como: Eu (conta Google) Quem tem acesso: Qualquer pessoa
5

Colar o URL aqui

Cole o URL do Web App na barra de sincronização (visível só para o Admin) e clique Ligar.
